Crayola Colors API

Get the HEX, RGB, and HSB values of your favorite crayons!

Request All Colors

GET https://ttesori-colors-api.herokuapp.com/api

Request Colors with Keyword

GET https://ttesori-colors-api.herokuapp.com/api/:query

Example

GET https://ttesori-colors-api.herokuapp.com/api/yellow
